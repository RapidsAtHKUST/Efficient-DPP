# gpuqp_opencl

/*
Source code for gpuqp on OpenCL
May, 2015
*/

This is the source code for the gpuqp (builded by He Bingsheng et. al in 2008) on OpenCL platform.

*INSTALL*
- OS and hardware: To run the code, please check whether your devices support OpenCL API. In this case, we use OpenCL 1.1. 
- To run the code, we need to install the SDK for the GPUs ( Nvidia, AMD, etc)  or coprocessors (MIC). For Nvidia, please pay attention 
  that only versions before CUDA 4.2 give supports to OpenCL.
